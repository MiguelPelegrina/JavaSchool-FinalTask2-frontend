Excel parser

Instructions

1. This excel parser allows you to choose an excel file (.xlsx), see all the values of it in a table and then import those books into the database.

2. All genres used in the excel need to exist before importing books with those genres.

3. Every book will be added. No book will be updated and no book with apparent same data will be ignored.
