Excel importer

Instructions

1. This importer allows you to choose an excel file, see all the values of it in a table and then import those books into the database.

2. Currently supported extensions:.

  • .xlsx

3. All new genres used in the excel will be created.

4. All new formats used in the excel will be created.

5. Books are identified by their ISBN. This has the following consequences:

  • If the ISBN is not assigned yet, the book will be added.
  • If the ISBN is already assigned, the book will be updated.
